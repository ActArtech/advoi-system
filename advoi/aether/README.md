# aether/

Portfolio manager and venture architect. Owns strategic context across ventures, assets, and initiatives.

## Purpose

- **Portfolio view** — active ventures, bets, and resource allocation
- **Venture architecture** — structure new initiatives from voice briefs
- **Strategic synthesis** — connect ontology definitions to real-world execution

## Boundaries

| In scope | Out of scope |
|----------|--------------|
| Venture lifecycle, portfolio health | Voice transport (→ `voice/`) |
| Strategic framing for decisions | Tactical crew work (→ `squads/`) |
| Cross-venture pattern recognition | Document ingestion (→ `ingestion/`) |

## Key Concepts

- **Venture** — bounded initiative with goals, constraints, and optionality
- **Portfolio** — collection of ventures with capital/attention allocation
- **Architect brief** — structured output feeding `decision/` frames
